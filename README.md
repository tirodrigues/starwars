# StarWars
API REST para cadastro de planetas desafio B2W.

## Tecnologias

-JAVA 8
-SPRING BOOT
-MONGODB

## RUN MAVEN
Com o banco devidamento configurado em "/starwars/src/main/resources.application.properties"

Executar: ``` mvn spring-boot:run ``` 

## Documentação
http://localhost:8080/swagger-ui.html
